# Кафе - Система управления

Полнофункциональное приложение на GTK3 для Linux для управления меню кафе, корзиной заказов и интеграции с платежным терминалом.

## Возможности

- **Иерархическое меню**: древовидная структура до 7 уровней вложенности
- **3 вкладки**: Меню | Корзина | Редактирование
- **Навигация**: раскрытие/сворачивание категорий по клику/тапу
- **Корзина заказов**: добавление товаров, удаление, автоматический подсчет суммы
- **Оплата**: отправка итоговой суммы на банковский терминал
- **Редактирование меню**: добавление категорий и товаров через интерфейс
- **Контекстное меню**: правый клик на категорию для быстрого добавления подкатегории
- **Полноэкранный режим**: оптимизация для сенсорных экранов
- **Светлая тема**: крупные элементы интерфейса (2x размер) для удобного управления
- **Сенсорный экран**: все кнопки и поля увеличены для работы на тачскрине

## Использование

### Вкладка "Меню"

1. **Просмотр категорий**: В дереве отображаются категории (Кофе, Чай, Напитки, Десерты)
2. **Раскрытие категорий**: 
   - Кликните на стрелку слева от категории
   - Или дважды кликните на название
3. **Выбор позиции**: Кликните на название напитка, чтобы увидеть его цену
4. **Добавление в корзину**: 
   - Дважды кликните на товар, или
   - Выберите товар и нажмите кнопку "Добавить в корзину"

### Вкладка "Корзина"

1. **Просмотр заказа**: Отображаются добавленные товары с количеством и суммой
2. **Удаление товара**: Выберите товар и нажмите "Удалить выбранное"
3. **Итоговая сумма**: Отображается внизу панели (крупным шрифтом)
4. **Оплата**: Нажмите кнопку "Оплатить" для отправки суммы на терминал

### Вкладка "Редактирование"

1. **Добавление категории**: Оставьте поле "Цена" пустым
2. **Добавление товара**: Укажите название и цену в рублях
3. **Выбор родителя**: Используйте выпадающий список для выбора родительской категории
4. **Быстрое добавление**: Правый клик на категории в меню → автоматически переходит в редактирование

### Полноэкранный режим

- Кнопка "Полный экран" в верхней панели
- Оптимизировано для сенсорных экранов кассы

## Особенности реализации

- **Глубина вложенности**: до 7 уровней категорий
- **Иерархия**: GtkTreeStore для древовидного отображения
- **Корзина**: массив до 50 позиций, автоматическое суммирование
- **Редактирование**: правый клик для быстрого добавления подкатегорий
- **Платежный терминал**: отправка суммы через system() команду
- **Сенсорный интерфейс**: все элементы увеличены для тачскрина
- **Полноэкранный режим**: кнопка переключения
- **C11 compliant**: чистый код без предупреждений

## Интеграция с платежным терминалом

Приложение отправляет сумму заказа через стандартный вывод. Для интеграции с конкретным терминалом:

1. Найдите функцию `on_checkout_clicked` в main.c
2. Замените строку с `system(command)` на вызов API вашего терминала
3. Или используйте pipe для перенаправления вывода

## Разрешение экрана

Рекомендуемое разрешение: 1600x1200 или выше для оптимального отображения enlarged UI.

## Горячие клавиши

- **Двойной клик** на товар → добавить в корзину
- **Правый клик** на категорию → добавить подкатегорию
- **Enter** в полях редактирования → быстрое добавление
