# Кафе - Иерархическое меню с корзиной

Приложение на GTK3 для Linux для управления иерархическим меню напитков и обработки заказов на кассе с сенсорным экраном.

## Возможности

- **Иерархическая структура**: древовидное меню с категориями и подкатегориями
- **Навигация**: раскрытие/сворачивание категорий по клику/тапу
- **Добавление элементов**: создание новых категорий и позиций через интерактивную форму
- **Корзина заказов**: добавление товаров, управление количеством, удаление позиций
- **Оплата**: отправка итоговой суммы на банковский терминал
- **Полноэкранный режим**: кнопка для переключения в полноэкранный режим
- **Светлая тема**: приятный светлый интерфейс с синими акцентами
- **Сенсорный экран**: крупные элементы интерфейса для удобного управления

## Требования

- Linux
- GCC с поддержкой C11
- GTK3 development libraries
- pkg-config

## Использование

### Навигация по меню

1. **Просмотр категорий**: В дереве отображаются категории (Кофе, Чай, Напитки, Десерты)
2. **Раскрытие категорий**: Кликните на стрелку слева от категории или дважды кликните на название
3. **Выбор позиции**: Кликните на название напитка, чтобы увидеть его цену

### Работа с корзиной

1. **Добавление в корзину**: 
   - Дважды кликните на товар в меню, или
   - Выберите товар и нажмите кнопку "Добавить в корзину"
2. **Просмотр корзины**: В правой панели отображаются добавленные товары с количеством и суммой
3. **Удаление из корзины**: Выберите товар в корзине и нажмите "Удалить выбранное"
4. **Итоговая сумма**: Отображается внизу правой панели

### Оплата заказа

1. Нажмите кнопку "Оплатить"
2. В диалоговом окне будет показана сумма к оплате
3. Подтвердите отправку на терминал
4. После успешной оплаты корзина очищается

### Добавление новых элементов

1. **Название**: Введите название нового элемента (например, "Капучино")
2. **Цена** (опционально): Введите цену в рублях. Если оставить пустым - элемент будет категорией
3. **Родительская категория**: Выберите, в какую категорию добавить элемент
   - "(корневой уровень)" - добавить как корневую категорию
   - Названия категорий - добавить в выбранную категорию
4. **Кнопка "Добавить"**: Нажмите для добавления элемента в меню

### Полноэкранный режим

- Нажмите кнопку "Полный экран" в верхней панели для переключения режима

## Особенности реализации

- Использует GtkTreeStore для отображения иерархических данных
- Корзина заказов с возможностью добавления/удаления товаров
- Автоматический подсчет итоговой суммы
- Диалоговое окно подтверждения оплаты
- Рекурсивное создание и отображение древовидной структуры
- Динамическое обновление интерфейса при добавлении новых элементов
- Автоматическое определение категорий (цена=0) и товаров (цена>0)
- Поддержка неограниченной глубины вложенности категорий

## Интеграция с платежным терминалом

Приложение отправляет сумму заказа на стандартный вывод (stdout), что позволяет:
- Перенаправить вывод в файл для логирования
- Использовать pipe для интеграции с внешними системами
- Модифицировать код для прямого взаимодействия с API терминала

Для интеграции с конкретным терминалом modify the `on_checkout_clicked` function to use the terminal's API or command-line tool instead of the echo command.
